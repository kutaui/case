# How to run

1. Clone the repository
2. Run the following command in the terminal

```bash
npm install
```

3. Run the following command in the terminal

```bash
npm run dev
```

4. Open the browser and go to the following URL

```bash
http://localhost:5173
```

5. Enjoy the app!

# How to run tests

1. Run the following command in the terminal

```bash
npm run test
```

2. Enjoy the tests!
